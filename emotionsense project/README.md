# EmotionSense AI

## Overview:

EmotionSense AI is an AI-powered emotion detection and analytics platform that analyzes user text and predicts emotional states using Natural Language Processing (NLP).

The application provides secure user authentication, personalized emotion history, data visualization, and real-time emotion analysis through an interactive web interface built with Streamlit.



## Features:

* User Registration and Login System
* Secure Password Hashing
* Transformer-Based Emotion Detection
* User-Specific Emotion History
* SQLite Database Integration
* Interactive Analytics Dashboard
* Emotion Distribution Visualization
* Real-Time Emotion Analysis
* Cloud Deployment with Streamlit

## Installation and Setup:

1. Navigate to the project directory:

cd “EmotionSense Project”

2. Install dependencies:

pip install -r requirements.txt

3. Run the application:

python -m streamlit run app.py



## Functionalities:

### Authentication

Users can create accounts and securely log in to access their personalized dashboard.

### Emotion Analysis

The application analyzes text input and predicts emotions using NLP models.

### Personalized Data Storage

Each user can only access their own emotion history and analytics.

### Analytics Dashboard

The dashboard provides:

* Total Entries
* Most Frequent Emotion
* Average Confidence Score
* Emotion Distribution Charts
